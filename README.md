  # Employee Management System

## Description 
This simple EMS command line application uses a mySQL database and the npm inquirer package to take in user selections and dynamically render results in the console.

## Installation
User needs to run npm install to download node modules and a package-lock.json. This provides the necessary dependencies that are already in the package.json.

## Usage
User starts the app by running node index.js in the console and following all the command line prompts.
